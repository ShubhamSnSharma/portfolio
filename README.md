# Shubham Sharma | The Learner Becoming The Builder

Welcome to the source code for my interactive, narrative-driven personal portfolio. 

🌐 **Live Website:** [internhuntt.vercel.app](https://internhuntt.vercel.app/) (Domain updating soon)

Unlike traditional static resumes, this portfolio is designed as a cinematic storytelling experience. It takes visitors on a journey from "The Learner" (absorbing knowledge) to "The Builder" (creating real-world impact), utilizing scroll-driven animations and highly interactive web graphics.

## ✨ Key Features

- **Cinematic Storytelling:** A 6-chapter narrative experience guided by high-performance smooth scrolling.
- **Interactive Canvas Graphics:** 
  - Physics-based particle explosions on scroll.
  - A dynamic, mouse-reactive data mesh visualization.
  - A glowing, scroll-tracked SVG topographical path.
- **Elite Performance:** Architected without heavy frontend frameworks (React/Vue) to achieve blazing fast load times and a near-perfect **99/100 Lighthouse Performance Score**.
- **Responsive Glassmorphism UI:** Modern, immersive, and fully responsive across mobile and desktop devices.
- **Accessible & SEO Optimized:** Perfect 100/100 scores in Best Practices, Accessibility, and SEO.

## 🛠️ Tech Stack

This project was built from the ground up focusing on web fundamentals to ensure maximum performance and granular control over the rendering pipeline:

- **Structure:** Semantic HTML5
- **Styling:** Vanilla CSS3 (Custom Variables, Flexbox/Grid, Glassmorphism, Advanced Media Queries)
- **Logic & Animation:** Vanilla JavaScript (ES6+), HTML5 `<canvas>` API, `requestAnimationFrame`
- **Smooth Scrolling:** [Lenis](https://lenis.studiofreight.com/) by Studio Freight

## 🚀 Running Locally

Because this project uses vanilla web technologies, running it locally is incredibly simple:

1. Clone the repository:
   ```bash
   git clone https://github.com/ShubhamSnSharma/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Run a local web server to prevent CORS issues with local assets. If you have Python installed:
   ```bash
   python3 -m http.server 8080
   ```
4. Open your browser and navigate to `http://localhost:8080`.
